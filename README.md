# Modern Calculator
A sophisticated desktop calculator application featuring a contemporary design and intuitive interface. Developed using the PyQt5 framework.

## Features
- **Modern Dark Theme**: A carefully designed dark interface that reduces eye strain during extended usage
- **Neon Green Accent**: Vibrant visual elements that enhance user experience and interface clarity
- **Basic Operations**: Addition, subtraction, multiplication, and division calculations
- **Advanced Functions**: Square root calculation capabilities
- **Intuitive Interface**: Clean and straightforward user interface design
- **Quick Access Functions**: Clear, Easy, and Delete buttons for efficient operation

## Requirements
- Python 3.7 or higher
- PyQt5 library
